# Video-FX
A real-time webcam effects tool built with p5.js for use in video calls, streaming, and OBS. It applies live visual filters and transformations to the camera feed, allowing for creative manipulation of how the webcam is displayed.

The tool includes the following modes and effects:

-ASCII-style rendering effect

-Screensaver DVD logo bounce

-Pixelation mode (low-resolution visual grid rendering)

-Rainbow Mode

-RGB color shifting and channel manipulation

-Video overlay / compositing mode

-Upside-down / inverted camera transformation

-Repetition filter
